# 宠物医院管理系统

#### 介绍
宠物医院管理系统

#### 软件架构
软件架构说明

springboot宠物管理系统
本系统共分为三个角色：系统管理员、医生、用户
框架：springboot、mybatis
数据库：mysql 5.7（注意版本不能为8）
功能模块：系统管理-用户管理、页面管理、角色管理
宠物管理-宠物列表、宠物健康史
预约管理-预约时间、医生时间
日常健康-健康指南、健康监测、健康标准
宠物档案
医院管理
